Um Sistema de Gerenciamento deLIVROS

Este projeto é uma aplicação de gerenciamento de livros, construída com uma stack moderna que inclui Angular 18 no front-end e Spring Boot no back-end, utilizando Java 21, MySQL, JPA, e Swagger/OpenAPI para documentação. A aplicação permite adicionar, listar, buscar, editar e excluir livros e autores.

No back-end, usamos MapStruct para mapeamento de dados e Lombok para simplificar o código. O banco de dados MySQL é gerenciado pelo JPA, garantindo um acesso eficiente e seguro aos dados. Com a integração do Swagger e OpenAPI, a API é documentada e fácil de explorar.

No front-end, a aplicação é desenvolvida em Angular 18, com uma interface moderna e responsiva, utilizando Flowbite e Tailwind CSS para estilização. Estes frameworks garantem que a experiência do usuário seja agradável e que a interface seja fácil de usar, independentemente do dispositivo.

Este projeto visa proporcionar uma experiência robusta e intuitiva para o gerenciamento de uma biblioteca de livros, permitindo que usuários realizem operações CRUD de forma simples e eficaz.
